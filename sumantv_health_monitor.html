Keep your Google Sheet as the master data source. Paste the Published CSV URL below — the tool fetches it live each time you sync.

▸ How to get the URL from Google Sheets
  1. Open your Google Sheet
  2. Go to File → Share → Publish to web
  3. Select the sheet tab you want
  4. Change format from Web page to Comma-separated values (.csv)
  5. Click Publish → copy the URL
  6. Paste it above and click Connect

⚠ The sheet must use the same column format as the sample CSV template.
